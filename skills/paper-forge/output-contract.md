# Output Contract

The canonical PaperForge output contract lives at:

```text
skills/paper-forge/references/output-contract.md
```

Quick reminders for this release:

- Obsidian reading notes now use titled filenames such as `01 - Problem, Prior Limitation, Actual Contribution.md`.
- Legacy bare filenames such as `00.md` through `05.md` are protected by default and are not auto-migrated.
- Zotero Local API remains a read-only input chain.
- Author names must remain Unicode-safe through metadata, workspace artifacts, and Obsidian Markdown.
- Zotero auto-tag writing is not implemented here.
