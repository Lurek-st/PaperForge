# Contribution Rubric

Separate novelty from naming, packaging, engineering integration, tuning, and presentation.

Answer:

1. What concrete problem does the paper solve?
2. Where do prior methods fail?
3. What is the causally meaningful intervention?
4. What remains if the proposed component is removed?
5. Which claims are directly supported and which are inferred?

Required sentences:

```text
This paper's core is: by ______, it solves ______, therefore it can ______.
If the authors' proposed ______ is removed, the remaining contribution is ______.
```

Do not summarize by table of contents. Use:

```text
problem -> prior limitation -> intervention -> mechanism change -> evidence -> conclusion boundary
```
