# Transfer Rubric

Use the Profile when available. Never equate relevance with deployability.

Required relevance layers:

- Theoretical relevance
- Algorithmic relevance
- System architecture relevance
- Industrial deployment relevance
- Commercial relevance
- Safety and reliability relevance

For industrial embodied intelligence, intelligent manufacturing, robotics, and control papers, inspect:

- Perception value
- Planning value
- Control value
- Robotics collaboration value
- Manufacturing automation value
- Real factory validation
- Real robot validation
- Latency
- Sensor and actuator uncertainty
- Safety
- Long-term stability
- Production rhythm
- Cost
- PLC / MES / SCADA integration
- Sim2Real gap
- Deployment, maintenance, and recovery
