# Paper Type Routing

Choose one primary type and optional secondary types.

Supported types:

- Theory
- Model / Algorithm
- Dataset
- Benchmark
- System
- Robotics / Embodied AI
- Industrial Deployment
- Hardware / Sensor
- Survey / Review

Theory emphasis:

- Assumptions
- Theorem scope
- Proof boundary
- Unspoken assumptions
- Whether practical conclusions are overextended

Model / Algorithm emphasis:

- Baseline fairness
- Ablation adequacy
- Data fairness
- Variance and repeated runs
- Benchmark overfitting
- Comparison against contemporary strong baselines

Robotics / Embodied AI / Industrial Deployment emphasis:

- Sensor noise
- Calibration
- Time synchronization
- Control latency
- Communication jitter
- Actuator precision
- Safety boundaries
- Fault recovery
- Long-run stability
- Sim2Real gap
- Real device evaluation
- Maintenance cost
- PLC / MES / SCADA integration

Dataset / Benchmark emphasis:

- Representativeness
- Leakage risk
- Annotation quality
- Distribution bias
- Metric validity
- Artificial benchmark effects
