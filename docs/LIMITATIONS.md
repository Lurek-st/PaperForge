# 限制 / Limitations

## 中文

Paper Forge 的限制：

- 不能保证论文是正确的。
- 不能替代专家同行评审。
- 可能误读公式、图像或低分辨率扫描。
- 不能分析无法访问的内容。
- 不能证明不存在数据泄漏。
- 不能证明工业可部署性。
- 结构验证不证明语义正确。
- Source locator 脚手架能提醒用户回到原文，但不能保证定位或引用一定正确；用户仍需核对原论文。
- 输出质量取决于来源材料、模型能力和用户审阅。
- Profile 能提升相关性分析，但不会让系统对用户或领域“全知”。

因此，Paper Forge 的输出应被视为结构化阅读辅助和审计线索，而不是最终事实裁决。

## English

Paper Forge limitations:

- It cannot guarantee that a paper is correct.
- It cannot replace expert peer review.
- It may misread formulas, figures, or low-resolution scans.
- It cannot analyze content it cannot access.
- It cannot prove the absence of data leakage.
- It cannot prove industrial deployability.
- Structural validation does not prove semantic correctness.
- Source locator scaffolding helps users return to the paper, but it does not guarantee that locators or citations are correct; users still need to check the original paper.
- Output quality depends on source material, model capability, and user review.
- A Profile improves relevance analysis but does not make the system personally omniscient.

Therefore, Paper Forge outputs should be treated as structured reading assistance and audit leads, not final factual verdicts.
